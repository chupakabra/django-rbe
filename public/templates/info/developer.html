Developer Info

If you are a software developer you might want to contribute to either RBE Connect or have other ideas on software that can improve the community of people advocating an RBE.

To support RBE Connect project you can find it on Github and contact me there. The programming language is python with the django web framework.

If you plan to build other tools it could be beneficiary to integrate those via OpenId. This allows users to authenticate via this page and you will easily get the users profile information, location data and entered skills. This page basically works as a database of people which provide certain datapoints which can be build upon in other applications.

An example project on how to integrate with RBE Connect will be soon available on github as well.
