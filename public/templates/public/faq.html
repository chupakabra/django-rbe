Q: What is RBE Connect?

Answer

RBE Connect is a website that makes communication and coordination between people advocating for a Resource Based Economy easier. It provides the possibility for users to record their location to find people nearby, record skills and interests such that the right person for the right question can be found and it also allows people to start initial communication via messages.

Q: Is the platform for everyone?

Answer

Yes, it is definitely okay if you join us even if you don't know exactly yet what an RBE is and what it aims for. It helps a lot though if you inform yourself and try to usnerstand what the general concepts of an RBE consists of. Once you have an overview the basics it is easier for you to decide where and how to help, given you like the concept.

Q: Why should I join yet another page?

Answer

This is a valid question and currently the honest answer is probably that there is no big reason to do so. Nevertheless this page aims to become the central instance for user profiles regarding active people advocating for an RBE. This means that the basic functionality is covered with this page and additional functionalities like Events, forums, lectures and everything one can think of will be surrounded to this page and also not all provided by one person.

Q: How can I build additional tools that use RBE Connect?

Answer

Building additional tools that make user of RBE Connect is great. If you thinking about doing so please contact me (Robert Kessler http://heleska.de) and we can talk about the technical details. In general you will get an Oauth2 access which allows you to query the API for user information and also update certain things. Note that the users need to agree that your tool has access to their profile and user specific data will only be given on consent. Some more abstract information can be obtained without the specific users though.
